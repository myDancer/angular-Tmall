# angular-Tmall

## 基于Angular.js实现的，仿天猫商城PC端webapp
用angular完全仿照Tmall

部分数据使用service代替 ，部分接口使用node.js+express+mongodb

完成了商品列表，搜索商品，商品详情，购物车等页面和交互

## 技术栈
- Angular.js: "^8.2.14"

## 开发坏境启动

1. `npm install`
2. `ng serve -o`

## 生产坏境部署

1. `npm run build`

2. 拷贝build文件夹至服务器即可
